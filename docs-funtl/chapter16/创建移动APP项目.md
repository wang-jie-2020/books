## 修改目录结构

![](/assets/Lusifer1520256185.png)

结构说明：

```
├─assets // 统一的资源文件目录
│  ├─apps // 项目所需的资源文件目录
│  │  ├─css
│  │  ├─img
│  │  └─js
│  └─global // 通用的资源文件目录
│      ├─css
│      │      mui.css
│      │      mui.min.css
│      ├─fonts // 字体文件目录
│      │      mui.ttf
│      └─js
│              mui.js
│              mui.min.js
```
