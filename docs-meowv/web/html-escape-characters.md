# HTML 转义字符表

| 字符                         | 十进制 | 转义字符 |
| ------------------------------ | ------ | -------- |
| "                              | &#34;  | &quot;   |
| &                              | &#38;  | &amp;    |
| <                              | &#60;  | &lt;     |
| >                              | &#62;  | &gt;     |
| 不断开空格(non-breaking space) | \&#160; | \&nbsp;   |
